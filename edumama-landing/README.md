# EduMama — Project scaffold

This folder was created by setup_edumama.ps1

## Files
- index.html — main landing + pilot sign-up
- grade-r.html — Grade R learning hub prototype
- css/style.css — main styles
- js/offline-tracker.js — local offline tracking and export
- service-worker.js — simple caching for offline (registers on pages)
- images/ — put hero.jpg and logo.png here

## How to run locally
1. Open this folder in VS Code.
2. Install Live Server extension (optional).
3. Right-click index.html → Open with Live Server.

## Notes
- Add your `images/hero.jpg` and `images/logo.png` to the images folder.
- Quizzes use Google Forms — paste the form URL into the Grade R page input.
